## How to Study Each Topic
1. Read **theory.py** completely — understand the concept, don't just skim
2. Try to solve **problems_easy.py** on your own FIRST before looking at solutions
3. Move to **problems_medium.py** only after you can solve easy ones without help
4. Run each file: `python theory.py` — they all have runnable examples

## Tips for JP Morgan SDE2
- Focus heavily on **Arrays, Strings, Hashing, Trees, and Graphs**
- Practice on LeetCode with the "JP Morgan" company tag
